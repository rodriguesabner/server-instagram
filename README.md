
# Instagram Server

## Tech Stack

Node, Express, Typescript, Awilix, Instagram Private API

## Run Locally

Clone the project

```bash
  git clone https://github.com/rodriguesabner/server-instagram
```

Go to the project directory

```bash
  cd server-instagram
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```
